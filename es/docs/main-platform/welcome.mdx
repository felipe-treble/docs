## Plataforma Principal

La plataforma principal de Treble es la plataforma que te permite crear y gestionar tus conversaciones outbound. Esta plataforma es la que vas a usar para crear tus conversaciones outbound y inbound.

En la plataforma principal vas a encontrar las siguientes secciones:

- **Conversaciones**: Esta es la vista principal de la plataforma. Aquí vas a encontrar todas las conversaciones que has creado.
- **Links de WhatsApp**: La sección de links de WhatsApp te permite crear links de WhatsApp que puedes usar para medir la interacción de tus clientes con tus links wa.me. 
- **Plantillas HSM**: La sección de plantillas HSM te permite crear plantillas de WhatsApp (HSM) que puedes usar para crear conversaciones outbound.
- **Métricas Generales**: Aquí vas a encontrar las métricas generales de tus conversaciones.
- **Documentación**: Treble tiene una documentación extensa que te permite entender cómo funciona nuestra API para que puedas integrar Treble con tus sistemas de información.
- **Configuración**: La sección de configuración te permite configurar tu cuenta de Treble. Aquí puedes configurar tu cuenta, tus preferencias, tus notificaciones, etc.
