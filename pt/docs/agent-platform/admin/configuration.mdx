# Como gerenciar a visibilidade de modelos HSM?

A partir da seção **Configuração > Preferências de HSM**, você pode controlar quais equipes de agentes têm acesso a cada modelo de mensagem HSM. Esta funcionalidade permite organizar e restringir o uso de modelos de acordo com as necessidades de cada grupo, garantindo que cada equipe veja apenas os modelos relevantes para sua operação.

## Quais informações são mostradas?

Na tabela, você encontrará uma lista de todos os modelos HSM disponíveis em sua conta, junto com os seguintes dados:

- **Modelo HSM:** Nome do modelo e um fragmento do conteúdo que contém.
- **Categoria:** Classificação atribuída ao modelo (por exemplo, "Marketing").
- **Permissões:** Equipes que têm visibilidade e acesso para utilizar o modelo.
- **Data da última modificação:** Última vez que o modelo foi atualizado ou editado.

> Os modelos que aparecem como "Nenhuma das equipes" não estarão visíveis para nenhum agente até que permissões específicas sejam configuradas.

## Como editar as permissões de um modelo?

Para alterar a visibilidade de um modelo:

1. Clique na coluna de **Permissões** na linha do modelo que deseja editar.
2. Um menu com todas as equipes disponíveis será exibido.
3. Marque ou desmarque as equipes às quais deseja dar acesso.
4. Clique em **Salvar alterações** para aplicar a configuração.

> ✅ Você pode selecionar a opção "Todas as equipes" para habilitar o uso global de um modelo.

## Para que serve esta funcionalidade?

Esta ferramenta é útil se você tem diferentes equipes usando o Treble para propósitos distintos (vendas, suporte, cobrança, etc.) e quer evitar confusões no momento de selecionar modelos. Ao atribuir permissões específicas, você garante uma experiência mais organizada e relevante para os agentes.

---

Você tem modelos duplicados ou modelos que já não usa? Você pode limpá-los e depois atualizar as permissões para manter sua conta organizada.
