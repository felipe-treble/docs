O Aplicativo Móvel treble permite que os agentes pesquisem usuários e contatos, visualizem o histórico de mensagens e enviem HSMs para usuários para iniciar conversas com valores personalizados. Com o aplicativo no celular, os agentes recebem notificações push quando há novas mensagens ou conversas.

Ao receber uma resposta, as conversas aparecerão na lista de chat. A interação é semelhante à plataforma, permitindo enviar imagens e arquivos, que podem ser baixados ou adicionados a partir do telefone.

Além disso, permite transferir conversas para equipes específicas ou agentes de vendas, ordenar mensagens de acordo com suas necessidades e gerenciar seu status ativo ou inativo na plataforma.

<Note>É importante que os agentes tenham o aplicativo instalado em seu celular para interagir com os clientes. Além disso, os agentes precisam ter uma senha atribuída para poder fazer login no aplicativo.</Note>

# Como baixar o aplicativo treble?

Você pode baixar o aplicativo treble a partir da página <a class="article-link" href="http://sales.treble.ai"><p>sales.treble.ai</p><Icon icon="arrow-up-right-from-square" iconType="solid" /></a>. Para instalá-lo, basta abrir esta página no navegador do seu dispositivo móvel.

A instalação varia dependendo do seu sistema operacional e navegador. Aqui estão os passos para instalar o treble:

### Para sistema iOS ou navegador Safari:

1. Acesse <a class="article-link" href="http://sales.treble.ai"><p>sales.treble.ai</p><Icon icon="arrow-up-right-from-square" iconType="solid" /></a> a partir do navegador Safari do seu telefone
