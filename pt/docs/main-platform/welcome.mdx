## Plataforma Principal

A plataforma principal do Treble é a plataforma que permite criar e gerenciar suas conversas outbound. Esta plataforma é o que você usará para criar suas conversas outbound e inbound.

Na plataforma principal, você encontrará as seguintes seções:

- **Conversas**: Esta é a visualização principal da plataforma. Aqui você encontrará todas as conversas que criou.
- **Links do WhatsApp**: A seção de links do WhatsApp permite criar links do WhatsApp que você pode usar para medir a interação de seus clientes com seus links wa.me.
- **Templates HSM**: A seção de templates HSM permite criar templates do WhatsApp (HSM) que você pode usar para criar conversas outbound.
- **Métricas Gerais**: Aqui você encontrará as métricas gerais de suas conversas.
- **Documentação**: O Treble tem uma documentação extensa que permite entender como nossa API funciona para que você possa integrar o Treble com seus sistemas de informação.
- **Configurações**: A seção de configurações permite configurar sua conta Treble. Aqui você pode configurar sua conta, suas preferências, suas notificações, etc.
