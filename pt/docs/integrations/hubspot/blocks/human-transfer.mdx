# Transferência para Humano

Esta funcionalidade é fundamental quando você deseja transferir a conversa para um agente. À medida que você constrói a conversa, se em algum momento desejar que ela seja recebida por um agente, você deve selecionar "transferência para humano" e escolher o grupo para o qual deseja transferir.

## Como utilizar o bloco de transferência para humano?

1. Faça login no Treble.ai em: <a class="article-link" href="https://app.treble.ai"><p>app.treble.ai</p><Icon icon="arrow-up-right-from-square" iconType="solid" /></a>
2. Vá para **Criar Conversa**
3. Clique no botão <Icon icon="square-plus" iconType="solid" /> para exibir o menu de blocos
4. Role até encontrar o menu de **Transferência para Humano** e dentro, selecione o bloco de **Transferência para Agente no HubSpot**
5. Clique no bloco de **Transferência para Agente no HubSpot** para adicioná-lo à sua conversa
6. Mova o bloco de **Transferência para Agente no HubSpot** para a posição desejada no seu fluxo
7. Conecte o bloco de **Transferência para Agente no HubSpot** com o bloco de **Pergunta**

<img src="/images/human-transfer-1.gif" alt="Conversa do Treble" />

8. Depois de adicionar o bloco de **Transferência para Agente no HubSpot**, clique duas vezes no bloco para exibir o menu de configuração
9. Você poderá selecionar a equipe para a qual deseja transferir a conversa

Este guia contém os passos específicos para configurar o bloco de transferência para humano no Treble relacionado ao HubSpot. No entanto, antes de continuar, recomendamos ler o guia do Bloco de Transferência para Agente do Treble, já que conceitos como Equipes, Agentes, etc. são os mesmos para ambos os blocos.

Uma vez que você tenha configurado o bloco de transferência para humano, sua equipe de agentes poderá responder a essas conversas diretamente do HubSpot. Para isso, você deve seguir os seguintes passos:

1. Faça login no HubSpot
2. Vá para **Caixa de Entrada**
3. Procure por "WhatsApp" na barra lateral esquerda
4. Ao clicar em "WhatsApp", você verá as conversas transferidas para sua equipe como agente
