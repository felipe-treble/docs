# Enviar uma Conversa do Treble do HubSpot para um Contato

Além de fazer envios em massa através de Workflows do HubSpot, o Treble também possui a funcionalidade de fazer envios individuais a partir do perfil do cliente no HubSpot.

Para realizar um envio a uma pessoa, você deve primeiro ir ao perfil do cliente que deseja contatar.

<img src="/images/deploy-1.png" alt="Conversa do Treble" />

Dentro do perfil, você encontrará o ícone do Treble. Tudo o que você precisa fazer é clicar nele, selecionar a conversa que deseja enviar e clicar em implantar.

<img src="/images/deploy-2.gif" alt="Conversa do Treble" />

<Warning>
    Lembre-se que só podemos enviar conversas outbound. Embora você encontre TODAS as conversas nesse ícone, só poderá enviar conversas outbound.
</Warning>
