# Workflows do HubSpot

No Treble, as conversas dentro de Workflows no HubSpot são uma ferramenta essencial para automatizar e personalizar a comunicação com clientes e leads. Ao integrar essas conversas, você pode criar interações automáticas baseadas em ações específicas dos usuários, garantindo uma atenção rápida e eficiente. Isso não só melhora a experiência do cliente, mas também otimiza seus processos internos para aumentar a produtividade e as conversões.

## Use os workflows do HubSpot para automatizar os envios de conversas

- Crie workflows baseados em contatos, empresas ou negócios e utilize os gatilhos que preferir
- Programe o envio de mensagens baseado em uma propriedade de contato, como a data de aniversário ou a data de fechamento do negócio, utilizando o tipo de workflow "Propriedade de data de contato"

## Como implantar uma conversa de WhatsApp (Treble) por Workflows do HubSpot

<Steps>
    <Step title="Ir para Workflows">
        A primeira coisa que você deve fazer é ir para HubSpot → Automações → Workflows

        <img src="/images/workflows-1.png" alt="Workflows do HubSpot" />
    </Step>
    <Step title="Criar novo workflow">
        Ao selecionar Workflows, você pode criar um do zero.

        <img src="/images/workflows-2.gif" alt="Workflows do HubSpot" />

        <Info>
            Os workflows podem ser criados para contatos, negócios, tickets e outros itens.
        </Info>
    </Step>
    <Step title="Configurar workflow">
        Ao clicar na caixa de texto, aparece uma barra lateral esquerda. Ali podemos selecionar "Quando os critérios do filtro são atendidos". Esta opção nos permite criar regras baseadas nos valores das propriedades.

        <img src="/images/workflows-3.png" alt="Workflows do HubSpot" />
    </Step>
    <Step title="Configurar critérios">
        Em seguida, selecionamos todas as propriedades que determinam os critérios que os clientes devem atender para fazer um envio. Quando tivermos os critérios prontos, devemos clicar no ícone do <Icon icon="plus" />

        <img src="/images/workflows-4.png" alt="Workflows do HubSpot" />
    </Step>
    <Step title="Adicionar ação do Treble">
        O que resta agora é selecionar a ação do Treble, escolher a conversa que queremos enviar e especificar o número a partir do qual o envio será feito.

        1. Ao clicar no botão <Icon icon="plus" />, será exibido um painel à esquerda com a lista de ações.
        2. Role até encontrar a seção de "Aplicativos Integrados"
        3. Selecione a opção "Treble: Integração WhatsApp"
        4. Uma vez que você clique na opção, serão exibidos dois campos:
            - "Conversa": Selecione a conversa do Treble que você quer que seja enviada. Esta conversa deve estar criada no Treble.
            <Tip> Conversas no Treble são fluxos de mensagens que você pode criar para automatizar o envio de mensagens aos seus clientes. Se você não sabe como criar uma conversa, pode consultar o seguinte artigo: <a class="article-link" href="/pt/docs/intro-to-treble/outbound-convo"><p>Criar conversa outbound</p><Icon icon="arrow-up-right-from-square" iconType="solid" /></a> </Tip>
            - "Número": No Treble, você pode ter múltiplos números de WhatsApp. Selecione o número a partir do qual você quer que a conversa seja enviada.
        5. Clique em "Salvar" para finalizar a configuração do workflow.

        <img src="/images/workflows-5.gif" alt="Workflows do HubSpot" />
    </Step>
    <Step title="Confirmar configuração do workflow">
        Confirme que o workflow está pronto e clique em "Salvar" para finalizar a configuração. Você pode ver a seguinte imagem para entender como o workflow ficaria no HubSpot.

        Como passo final, você pode publicar o workflow para que ele seja executado clicando no botão "Revisar e Publicar"

        <img src="/images/workflows-6.png" alt="Workflows do HubSpot" />
    </Step>
</Steps>

<Check>
    Parabéns! Você criou seu primeiro workflow no HubSpot para implantar uma conversa por WhatsApp com o Treble.
</Check>

<Info>
    LEMBRE-SE: Os workflows servem tanto para enviar mensagens em massa, quanto para enviar mensagens automatizadas.
</Info>
