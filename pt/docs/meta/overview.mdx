Meta, anteriormente Facebook, é uma empresa que tem como objetivo conectar pessoas através de seus produtos. WhatsApp é um dos seus produtos mais importantes. Treble é uma empresa que ajuda negócios a se conectarem ao WhatsApp usando a API do WhatsApp Business.

O que isso significa para você?

## Um pouco de história

Meta adquiriu o WhatsApp em 2014 por $19 bilhões de dólares. WhatsApp era um aplicativo de mensagens que já tinha 450 milhões de usuários ativos. Meta viu a oportunidade de usar o WhatsApp para conectar mais pessoas e expandir seu alcance. À medida que o aplicativo cresceu, Meta começou a ver a oportunidade de usar o WhatsApp para conectar empresas com seus clientes.

Em 2016, Meta lançou o WhatsApp Business, um aplicativo que permite às empresas se conectarem ao WhatsApp. WhatsApp Business é um aplicativo gratuito que permite às empresas enviar e receber mensagens. Tem mais funcionalidades que o aplicativo normal do WhatsApp e é otimizado para negócios. Por exemplo, permite às empresas terem um perfil comercial, que é um perfil desenhado especificamente para negócios.

Em 2018, Meta lançou a API do WhatsApp Business, uma API que permite às empresas se conectarem ao WhatsApp e enviar e receber mensagens com seus clientes. A API do WhatsApp Business é uma API de acesso restrito, disponível apenas para empresas que têm acesso direto da Meta ou que contam com um parceiro que tem acesso à API. Treble é um desses parceiros e utiliza a API do WhatsApp Cloud para permitir às empresas enviar e receber mensagens em grande escala.

Em 2022, Treble se tornou um dos parceiros da Meta para oferecer a API do WhatsApp Business. Ou seja, Treble é um BSP (Business Service Provider) que permite às empresas se conectarem ao WhatsApp e enviar e receber mensagens com seus clientes.
