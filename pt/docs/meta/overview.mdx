A Meta, anteriormente Facebook, é uma empresa que visa conectar pessoas através de seus produtos. O WhatsApp é um de seus produtos mais importantes. O Treble é uma empresa que ajuda negócios a se conectarem ao WhatsApp usando a API Business do WhatsApp.

O que isso significa para você?

## Um pouco de história

A Meta adquiriu o WhatsApp em 2014 por $19 bilhões. O WhatsApp era um aplicativo de mensagens que já tinha 450 milhões de usuários ativos. A Meta viu a oportunidade de usar o WhatsApp para conectar mais pessoas e expandir seu alcance. À medida que o aplicativo crescia, a Meta começou a ver a oportunidade de usar o WhatsApp para conectar empresas com seus clientes.

Em 2016, a Meta lançou o WhatsApp Business, um aplicativo que permite que empresas se conectem ao WhatsApp. O WhatsApp Business é um aplicativo gratuito que permite que empresas enviem e recebam mensagens. Ele tem mais funcionalidades que o aplicativo normal do WhatsApp e é otimizado para empresas. Por exemplo, permite que as empresas tenham um perfil empresarial, que é um perfil especificamente projetado para empresas.

Em 2018, a Meta lançou a API do WhatsApp Business, uma API que permite que empresas se conectem ao WhatsApp e enviem e recebam mensagens com seus clientes. A API do WhatsApp Business é uma API de acesso restrito, disponível apenas para empresas que têm acesso direto da Meta ou que têm um parceiro que tem acesso à API. O Treble é um desses parceiros e usa a API Cloud do WhatsApp para permitir que empresas enviem e recebam mensagens em escala.

Em 2022, o Treble se tornou um dos parceiros da Meta para oferecer a API do WhatsApp Business. Ou seja, o Treble é um BSP (Business Service Provider) que permite que empresas se conectem ao WhatsApp e enviem e recebam mensagens com seus clientes.
