## Entendendo a API do WhatsApp Business
A API do WhatsApp Business é uma plataforma que permite às empresas se conectarem ao WhatsApp e enviarem e receberem mensagens com seus clientes. Seria impossível entender o Treble sem compreender a API do WhatsApp Business. Por essa razão, é necessário entender todo o ecossistema do WhatsApp. O WhatsApp tem três produtos principais.

<img src="/images/plataformas.png" alt="WhatsApp, WhatsApp Business e API do WhatsApp Business" />

### Aplicativo WhatsApp
Este é o aplicativo que você, como usuário, tem no seu telefone. É um aplicativo gratuito que permite enviar e receber mensagens. Este aplicativo de mensagens é usado por 2 bilhões de pessoas em todo o mundo e permite que os usuários conversem com seus amigos e familiares.

### Aplicativo WhatsApp Business
Este é o aplicativo que as empresas usam para se comunicar com seus clientes. É um aplicativo gratuito que permite às empresas enviar e receber mensagens. Tem mais funcionalidades que o aplicativo normal do WhatsApp e é otimizado para negócios. Por exemplo, permite às empresas terem um perfil comercial, que é um perfil desenhado especificamente para negócios.

### API do WhatsApp Business
Esta é a API que as empresas usam para se conectar ao WhatsApp e enviar e receber mensagens em grande escala. É uma API de acesso restrito, disponível apenas para empresas que têm acesso direto da Meta ou que contam com um parceiro que tem acesso à API. Treble é um desses parceiros e utiliza a API do WhatsApp Cloud para permitir às empresas enviar e receber mensagens em grande escala.

<Note>Ao usar a API do WhatsApp Business, tenha em mente algumas limitações. Estamos acostumados a ter um aplicativo, seja do WhatsApp ou do WhatsApp Business em nosso telefone. No entanto, quando usamos a API do WhatsApp Business, não temos um aplicativo em nosso telefone. Além disso, não podemos fazer chamadas ou criar grupos no WhatsApp.</Note>

Vamos olhar mais de perto a API do WhatsApp Cloud.

## API do WhatsApp Business
Como mencionado anteriormente, Treble é um parceiro da API do WhatsApp Business. Isso significa que Treble tem acesso à API do WhatsApp Cloud. Com esta API, podemos gerar uma conta comercial do WhatsApp (WABA) para nossos clientes. Se você já recebeu uma mensagem de uma empresa no WhatsApp, você interagiu com a API do WhatsApp Business. Provavelmente você reconhece essas contas, aqui está um exemplo:

<img src="/images/ejemplo-wa.png" alt="Exemplo de uma conta comercial do WhatsApp" />

Como você pode ver, a conta comercial é uma conta verificada. Isso significa que a empresa tem um perfil verificado no WhatsApp. Isso ajuda a empresa a gerar mais confiança e se destacar de outros negócios no WhatsApp.

Você também deve ter notado que a mensagem contém um link clicável. Estas mensagens são especiais da API do WhatsApp Business e são chamadas de Mensagens Interativas. Mais adiante falaremos sobre elas.

### Fluxo simples de uma mensagem no WhatsApp
Vamos ver como funciona um fluxo simples de mensagens no WhatsApp. Suponhamos que uma empresa queira enviar uma mensagem a um cliente. A empresa tem uma conta comercial do WhatsApp (WABA) e o cliente usa o aplicativo normal do WhatsApp. A empresa quer enviar uma mensagem ao cliente.

1. A empresa envia uma mensagem ao cliente usando a API do WhatsApp Business.
2. O cliente recebe a mensagem da empresa.
3. O cliente responde à mensagem.
4. A empresa recebe a resposta do cliente.

<img src="/images/flujo.png" alt="Fluxo simples de uma mensagem no WhatsApp" />

Como você pode ver, o fluxo de mensagens é bastante simples. A empresa envia uma mensagem, o cliente a recebe e responde, e a empresa recebe a resposta.

Ótimo! A empresa pode enviar e receber mensagens. Mas o que acontece se a empresa quiser enviar uma mensagem a múltiplos clientes? Ou se quiser responder com uma mensagem específica de acordo com a resposta do cliente? Ou mesmo se quiser atualizar as informações do cliente em seu CRM?

Aqui é onde a API do WhatsApp Business se torna mais interessante.

Se uma empresa quisesse fazer isso, precisaria construir sistemas sofisticados que gerenciem o fluxo de mensagens. É aqui que entra o Treble. Treble ajuda as empresas a construírem sistemas avançados que gerenciam a mensagem do WhatsApp sem complicações, fornecendo uma plataforma web simples para criar e administrar conversas.

## Mensagens na API do WhatsApp Business
Imagine que você é uma empresa com acesso à API do WhatsApp Business. Soa bem, não é? Agora você pode enviar e receber mensagens de seus clientes. Mas, o que isso realmente significa?

Se você tivesse todos os números de telefone dos usuários do WhatsApp no mundo, poderia enviar-lhes uma mensagem a todos? E quanto ao conteúdo da mensagem? Poderia enviar um link a todos eles para que experimentem sua nova startup: "Insulina para cães gordinhos"?

<img src="/images/perritos-gordos.png" alt="Insulina para cães gordinhos" />
Bem, a resposta é não. A API do WhatsApp Business tem restrições sobre o que pode ser enviado e recebido. Vamos explorar essas restrições.

A Sessão do WhatsApp
Meta (empresa matriz do WhatsApp) regula estritamente a maneira como as empresas podem se comunicar com seus clientes através da API do WhatsApp Business. Isso ajuda a Meta a garantir que a plataforma do WhatsApp seja um espaço seguro para todos os usuários.

Imagine que você constantemente recebe mensagens de SPAM de uma empresa. Seria irritante, não é? Provavelmente você deixaria de usar o WhatsApp e convenceria seus amigos a mudarem para outra plataforma, como o Telegram. Meta quer evitar isso, por isso implementou certas restrições na mensagem empresarial.

Antes que uma empresa possa enviar uma mensagem a um usuário, este deve passar por um processo de verificação. Meta permite este processo mediante a criação de uma Mensagem Altamente Estruturada (HSM, por suas siglas em inglês), também conhecida como template.

Esta mensagem deve ser aprovada pela Meta e pode levar até 48 horas para ser aprovada.
